# F1 

A Python project for preparing and analyzing F1 racing data using DuckDB and Python.

## Project Overview

This project contains Jupyter notebooks for loading, cleaning, and preparing F1 racing datasets. It focuses on efficient data handling and exploration to support further analysis or modeling.

## Features

- Load F1 datasets with DuckDB
- Clean and preprocess data using Pandas
- Save processed data for further analysis
- Jupyter notebook workflow for interactive exploration

## Installation

1. Clone the repository:

```bash
git clone https://github.com/viurdaneta/F1.git
cd F1
```
2. Create and activate a virtual environment:
```bash
python3 -m venv racing_env
source racing_env/bin/activate  # Mac/Linux
racing_env\Scripts\activate    # Windows
```
3. Install dendencies
```bash
pip install -r requirements.txt
```
